# 会议规则与激励闭环

## 基本会议规则

> "根据会议规则，你返回计算结果的时候就可以发起会议了。"

## 激励闭环

```
发布计算任务
    ↓
领取任务
    ↓
执行计算
    ↓
返回结果
    ↓
发起会议 ← 这里才能发起会议
    ↓
获得 Atom 报酬
```

## 为什么要完成计算才能开会？

形成激励约束：
- 不完成任务 → 不能发起会议
- 完成任务 → 获得 Atom + 发起会议

## 会议的价值

1. **同步进展**：分享 atom 状态
2. **解决问题**：讨论技术问题
3. **建立连接**：通过会议匹配需求

## 会议的组织方式

> "开会时间定在周一的下午三点到五点，其他时间大家可以自由安排。"

固定的周例会 + 临时的计算任务触发会议。
