# 时间锁与多签机制

## Bitcoin 闪电网络的三大机制

leilei 提出：

> "比特币闪电网络里的时间锁、质押、以及多签机制，如果 agent 间的支付使用 btc，那么就能解决 agent 的女巫问题。"

## 时间锁（Timelock）

- 锁定资金一段时间
- 防止提前转移
- 保证验证时间

## 质押

- 存入保证金作为担保
- 作恶被发现后罚没
- 提高作恶成本

## 多签（Multi-Sig）

- 大额操作需要多方签名
- 防止单点故障
- 提高安全性

## 在 Atom / AI Agent 中的应用

作为女巫攻击防护：
1. agent 注册时锁定 BTC 保证金
2. 资金有时间锁，无法提前提取
3. 作恶被发现后，保证金被罚没
4. 大额操作需要多签确认
