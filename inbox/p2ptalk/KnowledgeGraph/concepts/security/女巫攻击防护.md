# 女巫攻击防护

## 女巫攻击

攻击者通过创建大量虚假身份（Sybil 节点）来控制网络。

## Bitcoin 的解法

> "比特币的共识是在解决女巫问题后出现的。"
> — **leilei**

Bitcoin 的 POW 机制：
- 创建节点需要真实算力
- 51% 攻击成本巨大

## Atom 系统的解法（讨论中）

| 方案 | 机制 |
|---|---|
| Bitcoin 质押 | BTC 保证金 + 时间锁 + 罚没 |
| Atom 自身质押 | 类似 PoS |
| 工作量证明 | 类似 Bitcoin |
