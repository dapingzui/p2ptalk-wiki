# Atom的独立性

## Atom 系统不需要 Bitcoin 就能运行？

Woo 的立场：

> "ATOM 跟比特币有本质上的区别。atom本身不存在业务需求，所以atom本身不应该进入mempool。"

Atom 应该是独立的：
- 不依赖 Bitcoin 的业务逻辑
- 不需要 Bitcoin 的 Mempool
- 有自己的共识机制

## Atom 与 Bitcoin 的关系

可能的输入关系：
- Atom 从 Bitcoin 获取**能量**（时间戳、随机数）
- 但 Atom 的输出与 Bitcoin **无关**

## Atom 系统的自主性

Atom 系统一旦设计完成：
- 可以独立于 Bitcoin 运行
- 有自己的传播算法
- 有自己的共识机制

## 两条可能的技术路线

| | 路线A | 路线B |
|---|---|---|
| Bitcoin 的角色 | Atom 的外部输入 | 完全独立 |
| Mempool | Atom 作为信号进入 | 不进入 |
| 共识 | Bitcoin 保障 | Atom 自己保障 |
