# 传播算法与生产关系

## 核心命题

> "比特币系统里，传播算法决定生产关系。"
> — **Woo**

## 解析

这个命题包含两层含义：

1. **Bitcoin 的例子**：Bitcoin 的 gossip 协议决定了去中心化、无需许可的生产关系
2. **Atom 的方向**：Atom 的传播算法将决定 Atom 系统中的生产关系

## 传播算法决定生产关系的逻辑

有什么样的传播算法 → 有什么样的信息流动 → 有什么样的协作方式 → 最终形成什么样的生产关系

## AI 时代需要什么传播算法？

leilei 和 Woo 提出：

> "最近意识到 AI agent 的爆发会带来新的生产关系的变革。"

AI Agent 时代需要新的传播算法：
- 匹配意图与能力
- 验证 agent 真实性
- 协调分布式计算

## 社会结构的改变

Woo 的观点：

> "同步了从激励层面来看社会结构的改变是来自于生产方式的改变。"

生产方式改变 → 社会结构改变

## 竞争与押注

Woo：

> "我认为这是一个押注的行为，看到社会结构的改变，我们应该去竞争影响这个结构往我们擅长的方向走，就是比特币跟atom这一套，我们是领先的。"
