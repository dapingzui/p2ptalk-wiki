# 发布订阅机制

## Bitcoin 的发布订阅

Bitcoin 使用 Gossip Protocol（八卦协议）：
- 无差别广播
- 所有节点收到所有消息
- 无订阅关系

## Atom 的发布订阅

文博：

> "关于商品 offerbook 的传播完全继承比特币的发布订阅和中间会面机制。"

Atom 的发布订阅特点：
- 有订阅关系
- 通过中间会面转发
- 谓词验证过滤

## 发布订阅与匹配

发布订阅解决的问题：
- **意图广播**：我想要什么
- **能力广播**：我能提供什么

匹配 = 意图 + 能力的对接

## 中间会是发布订阅的增强

中间会面 = 发布订阅 + 谓词验证
- 不是所有订阅者都收到
- 而是满足谓词才转发
