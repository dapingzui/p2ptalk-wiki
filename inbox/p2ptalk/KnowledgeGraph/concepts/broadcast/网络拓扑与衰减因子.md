# 网络拓扑与衰减因子

## 胥亚朋的分析

> "关于 Atom 在一个良好的网络拓扑中的传递机制分析。
> 1. 收到 Atom 后会向每个 vLinksOut 中的地址进行传播。
> 2. 流量控制，除了第一个收到 Atom 的 CUser 对象，其他对象收到 Atom 以后会进行减半传播。"

## 衰减因子的影响

> "Atom 的传播主要受流量控制的影响，导致 Atom 的传播并不普遍和均匀，在随机信号源和随机路径下差异较大，中心 User 和中心 User 的 Atom 差异较小。Atom 很难跨区域传播。"

关键发现：
- Atom 传播**不普遍、不均匀**
- **中心节点聚集**更多 Atom
- 边缘节点难以获得 Atom
- 衰减因子导致跨区域传播困难

## 解决方案

> "调节衰减因子能显著改善全网流转情况，在分布不均的网络拓扑中更易形成区域不一致。"

## 局部共识 vs 全域共识

> "所以流量控制的作用应该单独进行思考，接下来需要考虑的是局部共识和全域共识之间的关系，进一步探究 Atom 的使用场景。"
> — **沄涣**
